# MyOllama 使用指南

MyOllama 是一款智能对话助手，支持文档问答、图片识别、语音输入等功能。

## 功能介绍

- **智能问答**：与 AI 聊天，获得智能回答
- **文档上传**：上传 PDF、Word 或文本文件进行问答
- **图片上传**：上传图片并针对图片提问
- **截图识别**：截取屏幕区域并用 AI 分析
- **语音输入**：使用麦克风语音输入文字
- **语音朗读**：让 AI 朗读回答内容

---

## 安装步骤

### 1. 安装 Python

确保已安装 Python 3.8 或更高版本。

### 2. 安装依赖

```bash
# 激活虚拟环境
.venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

---

## 启动应用

```bash
python app.py
```

打开浏览器访问：**http://localhost:5000**

---

## 首次使用配置

首次打开应用时，需要先配置 API Key。点击右下角的 ⚙️ 设置按钮：

1. 选择 API 模式（OpenAI 或 Anthropic）
2. 填写 API 地址和 Key
3. 点击保存

### 常用 API 配置示例

#### Ollama（本地部署）
- **地址**：`http://localhost:11434`

#### Minimax API（Anthropic 兼容模式）
- **地址**：`https://api.minimaxi.com/anthropic`
- **模型**：例如 `MiniMax-M2.7`

> ⚠️ 注意：地址末尾是 `/anthropic`，`langchain_anthropic` 会自动加 `/v1/messages`

#### Minimax API（OpenAI 兼容模式）
- **地址**：`https://api.minimaxi.com/openai/v1`
- **模型**：例如 `MiniMax-M2.7`

配置完成后即可开始使用。

## 使用方法

### 开始对话

1. 点击左侧侧边栏的 **+ 新对话**
2. 在输入框中输入问题
3. 按回车键或点击发送按钮

### 上传图片

**方式一：点击上传按钮**
1. 点击输入框下方的 🖼️ 图片按钮
2. 选择一张图片
3. 图片会显示在上方的图片栏中
4. 直接提问关于图片的问题

**方式二：拖拽上传**
1. 将图片文件拖拽到聊天区域任意位置
2. 聊天区域会变成蓝色高亮提示
3. 松开鼠标即可上传
4. 支持格式：JPG、PNG、GIF、WebP

### 上传文档

**方式一：点击上传按钮**
1. 点击 📤 上传文档按钮
2. 选择 PDF、Word 或文本文件
3. 等待处理完成（会有进度条显示）

**方式二：拖拽上传**
1. 将文档拖拽到聊天区域
2. 支持格式：PDF、DOCX、TXT
3. 等待处理完成后即可问答

### 语音输入

1. 点击 🎤 麦克风按钮开始录音
2. 对着麦克风说话
3. 说完后再次点击按钮停止录音
4. 语音会自动转为文字并发送

### 截图识别

1. 按 `Alt + A` 快捷键
2. 选择要截取的屏幕区域
3. 截图会自动上传并显示在图片栏中
4. 直接提问关于截图的问题

## 高级功能

### 对话管理

- **新对话**：点击左侧 **+** 按钮创建
- **切换对话**：点击左侧历史记录切换
- **删除对话**：鼠标悬停后显示删除按钮
- **对话分叉**：在历史消息处点击分支按钮创建新分支

### 文档问答

上传文档后，系统会根据问题自动决定披示层级：
- **摘要级**：询问"总结"、"概括"时返回 AI 生成的摘要
- **相关片段**：一般问题返回最相关的文档片段
- **完整内容**：询问"详细"、"完整"时返回更多内容

## 故障排除

### API 调用失败

1. 检查 API 地址是否正确
2. 检查 API Key 是否有效
3. 检查网络连接

### 语音无法识别

1. 确保浏览器有麦克风权限
2. 尝试使用 Chrome 或 Edge 浏览器

### 文档上传失败

1. 检查文件格式是否支持（PDF、DOCX、TXT）
2. 检查文件大小是否过大
