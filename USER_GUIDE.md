# MyOllama 使用指南

MyOllama 是一款智能对话助手，支持文档问答、图片识别、语音输入等功能。

## 功能介绍

- **智能问答**：与 AI 聊天，获得智能回答
- **文档上传**：上传 PDF、Word 或文本文件进行问答
- **图片上传**：上传图片并针对图片提问
- **截图识别**：截取屏幕区域并用 AI 分析
- **语音输入**：使用麦克风语音输入文字
- **语音朗读**：让 AI 朗读回答内容

---

## 安装步骤

### 1. 安装 Python

确保已安装 Python 3.8 或更高版本。

### 2. 安装依赖

```bash
# 激活虚拟环境
.venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```

---

## 启动应用

```bash
python app.py
```

打开浏览器访问：**http://localhost:5000**

---

## 首次使用配置

首次打开应用时，需要先配置 API Key。点击右下角的 ⚙️ 设置按钮：

1. 选择 API 模式（OpenAI 或 Anthropic）
2. 填写 API 地址和 Key
3. 点击保存

配置完成后即可开始使用。

---

## 使用方法

### 开始对话

1. 点击左侧侧边栏的 **+ 新对话**
2. 在输入框中输入问题
3. 按回车键或点击发送按钮

### 上传图片

**方式一：点击上传按钮**
1. 点击输入框下方的 🖼️ 图片按钮
2. 选择一张图片
3. 图片会显示在上方的图片栏中
4. 直接提问关于图片的问题

**方式二：拖拽上传**
1. 将图片文件拖拽到聊天区域任意位置
2. 聊天区域会变成蓝色高亮提示
3. 松开鼠标即可上传
4. 支持格式：JPG、PNG、GIF、WebP

### 上传文档

**方式一：点击上传按钮**
1. 点击 📤 上传文档按钮
2. 选择 PDF、Word 或文本文件
3. 等待处理完成（会有进度条显示）

**方式二：拖拽上传**
1. 将文档拖拽到聊天区域
2. 支持格式：PDF、DOCX、TXT
3. 等待处理完成后即可问答

### 截图

1. 按 **Alt + A** 或点击 📸 截图按钮
2. 拖动选择需要截取的区域
3. 截图会自动上传
4. 直接提问关于截图的问题

### 语音输入

1. 点击 🎤 麦克风按钮
2. 听到提示音后开始说话
3. 语音会自动转为文字
4. 再次点击停止录音

### 语音朗读

1. 点击 🔇 按钮开启 AI 语音朗读
2. AI 的回答会朗读出来
3. 再次点击关闭

### 切换 AI 模型

使用顶部的下拉菜单切换不同模型：
- 文本模型：qwen3:8b、qwen3:14b、deepseek-r1:8b
- 多模态模型：qwen3-vl:8b、qwen3.5:9b

上传图片时会自动切换到多模态模型。

## 实用技巧

- 对话内容会自动保存
- 可以创建多个对话
- 使用 "Fork" 从任意历史消息分叉新对话
- 拖拽上传支持图片和文档同时上传
- 按 **Alt + A** 快速截图
