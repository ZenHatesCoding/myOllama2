# PDF to Org 转换

这个 Skill 帮助你将 PDF 论文转换为 Org 格式，便于使用 Emacs/Org-mode 进行分析和笔记整理。

## 使用方法

### 步骤 1：读取 PDF 文件
使用 Read 工具读取你要分析的 PDF 文件内容。如果文件是扫描件或无法直接读取，请先使用 OCR 或其他方式转换为文本。

### 步骤 2：规划 Org 文档结构
根据 PDF 内容，规划 Org 文档的结构：

```
* 论文标题
** 作者信息
** 摘要
** 关键发现
** 方法论
** 结论
** 个人笔记
```

### 步骤 3：使用 Write 工具创建 Org 文件
使用 Write 工具将内容写入 .org 文件：

```bash
Write(path="output/paper_analysis.org", content="* 论文标题\n** 摘要\n...")
```

## 内置工具

| 工具 | 用途 |
|------|------|
| Read | 读取文件内容 |
| Write | 创建或覆盖文件 |
| Bash | 执行脚本 |
| Glob | 搜索文件 |
| Grep | 在文件中搜索 |

## 注意事项

- Org 模式使用 `*` 表示标题层级
- 使用 `**` 表示二级标题，以此类推
- 可以添加 `:PROPERTIES:` 进行元数据管理
